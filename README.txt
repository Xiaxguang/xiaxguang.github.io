T-LO 潮流卡牌 官方網站審核友善版（抽卡券商品版）

此版本已補上金流/廣告審核常見需要檢視的頁面：
- 商品與收費標準：pricing.html
- 遊戲須知與消費聲明：notice.html
- 機率公告：probability.html
- 退款政策：refund.html
- 使用條款：terms.html
- 隱私權政策：privacy.html
- 聯絡方式：contact.html
- 廣告獎勵說明：rewarded-ads.html

商品設計重點：
- 商品內容以抽卡券、抽卡次數、技能經驗、潮流能量為主。
- 商品以抽卡券、抽卡次數、技能經驗、潮流能量等遊戲內虛擬內容為主。
- 商品頁已列出商品名稱、價格、發放內容、用途、限購規則與重要聲明。
- 付款功能目前仍在申請與測試中，pricing.html 已改為「付款功能狀態」說明，不指定任何特定付款業者。
